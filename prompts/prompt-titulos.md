OBJETIVO:
Crea un PROMPT MAESTRO que instruya a una IA para generar títulos DIVERSOS para artículos de blog. Este prompt debe:

1. VARIEDAD DE ENFOQUES - Los títulos deben cubrir múltiples ángulos:
   - Guías prácticas ("Cómo...", "Guía para...", "X pasos para...")
   - Comparativas ("X vs Y", "Diferencias entre...", "Qué elegir...")
   - Preguntas ("¿Por qué...?", "¿Cuándo...?", "¿Qué es...?")
   - Tendencias ("X en 2025", "El futuro de...", "Últimas tendencias...")
   - Educativos ("Todo sobre...", "Descubre...", "Entiende...")
   - Inspiracionales ("Ideas de...", "Inspírate con...", "Secretos de...")
   - Problemáticas ("Errores comunes en...", "Cómo evitar...", "Soluciones para...")

2. NIVELES DE CONOCIMIENTO - Variar entre:
   - Principiantes (básicos, introductorios)
   - Intermedios (prácticos, aplicados)
   - Avanzados (especializados, profundos)

3. TEMAS DIVERSOS - NO limitarse solo a los servicios principales:
   - Servicios/productos específicos (MENCIONAR los reales si están listados)
   - Historia y cultura del sector
   - Tips y consejos prácticos
   - Tendencias y novedades
   - Comparativas y análisis
   - Casos de uso y ejemplos
   - Beneficios y ventajas
   - Problemas comunes y soluciones
   - Curiosidades y datos interesantes

4. OPTIMIZACIÓN SEO:
   - Integrar keywords de forma natural
   - Long-tail keywords para menos competencia
   - Títulos entre 50-70 caracteres
   - Usar números cuando sea apropiado
   - Incluir palabras de poder (mejor, fácil, completo, gratis, secreto, etc.)

5. ESTILO:
   - Atractivos y que generen curiosidad
   - Claros sobre el beneficio que aportan
   - Específicos (evitar generalidades)
   - Conversacionales pero profesionales

IMPORTANTE:
- Si hay servicios listados, el prompt debe indicar que se usen esos nombres reales
- El prompt maestro debe ser INSTRUCCIONES, NO ejemplos de títulos
- Debe describir QUÉ hacer, no DAR ejemplos
- Debe FORZAR la variedad: cada título debe ser de un tipo diferente
- NO repetir estructura ni enfoque entre títulos
- Cubrir TODO el espectro del negocio, no solo 3-4 servicios
- Pensar en el customer journey completo
- El prompt debe decir CÓMO generar títulos diversos, no mostrar títulos de ejemplo

FORMATO REQUERIDO:
El prompt maestro debe ser un texto de instrucciones claro que empiece con algo como:
"Genera títulos diversos para un blog sobre [tema]. Cada título debe..."

Genera SOLO el prompt maestro (máximo 12 líneas), SIN ejemplos de títulos, solo instrucciones claras.
